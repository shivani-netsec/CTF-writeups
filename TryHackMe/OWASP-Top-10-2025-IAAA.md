# OWASP Top 10 2025: IAAA Failures (AO1, AO7, AO9)


## Overview
This lab covers common web application security failures related to Identification, Authentication, Authorization, and Accounting (IAAA) as defined in OWASP Top 10 2025.

## Topics Covered
- A01: Broken Access Control
- A07: Authentication Failures
- A09: Logging & Alerting Failures


## Key Concepts Learned
- How improper authentication or authorization leads to data exposure
- Recognizing misconfigured servers and endpoints
- Basic techniques for testing access control flaws ethically


## Security Relevance
- Helps in understanding the most critical web vulnerabilities
- Essential knowledge for secure web development, penetration testing, and SOC monitoring
