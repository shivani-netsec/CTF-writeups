# Web Application Basics

## Topics Covered
- How web applications work
- HTTP and HTTPS
- Request and Response model
- Status codes
- URL structure
- Introduction to web vulnerabilities

## Key Concepts Learned
- How clients communicate with servers
- Understanding GET vs POST requests
- Importance of secure web communication (HTTPS)
- Common entry points for web attacks

## Security Relevance
Web applications are one of the most common attack surfaces. Understanding HTTP communication and basic web structure is essential for penetration testing and vulnerability assessment.
