# Active Directory Basics

## Overview
This lab introduced Active Directory (AD), a centralized system used to manage users, computers, and permissions in enterprise environments.

## Topics Covered
- Domain structure
- Users and groups
- Managing Users in AD
- managing computers in AD
- Group policy
- 
- Authentication (Kerberos basics)
- Domain Controller role
- Organizational Units (OUs)

## Tools Used
- Windows environment
- Active Directory concepts (GUI-based tools)

## Key Concepts Learned
- How centralized authentication works
- Importance of domain controllers
- Managing users and permissions in a network
- Role of Active Directory in enterprise security

## Skills Practiced
- Understanding domain-based environments
- Identifying user roles and privileges
- Recognizing how authentication systems operate

## Security Relevance
Active Directory is a primary target for attackers. Understanding AD is critical for detecting:
- Unauthorized access
- Privilege escalation
- Lateral movement in networks

## Reflection
This lab helped me understand how enterprise environments are structured and how authentication and access control are managed centrally.
