# Active Directory Basics

## Overview
This lab introduced Active Directory (AD), a centralized system used to manage users, computers, and permissions in enterprise environments.

## Topics Covered
- Domain structure
- Users and groups
- Managing Users in AD
- managing computers in AD
- Group policy
- trees,forest and trust
- Authentication (Kerberos basics)
- Domain Controller role
- Organizational Units (OUs)

## Key Concepts Learned
- How centralized authentication works
- Importance of domain controllers
- Managing users and permissions in a network
- Role of Active Directory in enterprise security
