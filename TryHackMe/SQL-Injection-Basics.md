# SQL Injection Basics

## Overview
This lab introduced SQL Injection techniques used to exploit vulnerabilities in web applications by manipulating database queries.

## Topics Covered
- SQL Injection 1: Input Box (Non-String)
- SQL Injection 2: Input Box (String)
- SQL Injection 3: URL Injection

## Concepts Learned
- How SQL queries interact with user input
- Difference between string and non-string injections
- How attackers manipulate input fields to bypass authentication

## Skills Practiced
- Identifying vulnerable input fields
- Testing basic SQL injection payloads
- Understanding authentication bypass techniques
