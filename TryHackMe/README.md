# Windows Fundamentals
this lab introduced core Windows operating system concepts important for cybersecurity and system administration.

## Topics Covered
- Windows file system structure
- NTFS permissions
- Windows Registry basics
- User accounts and groups
- System processes
- Event Viewer basics

## Tools Used
- Windows Command Prompt
- PowerShell (basic exposure)
- Event Viewer

## Key Concepts Learned
- Difference between local users and administrators
- Importance of file permissions
- Role of the Windows Registry
- How Windows logs events for monitoring

## Skills Practiced
- Navigating directories in Windows
- Viewing system information
- Understanding user privilege levels
- Identifying system services and processes
